3.注释我们使用以#开头的语句是注释，请务必注意，Python程序是大小写敏感的，如果写错了大小写，程序会报错。
4.为了防止乱码，文件开头一定要添加
```java
#!/usr/bin/env python
# -*- coding: utf-8 -*-
print u"中文测试"
```
